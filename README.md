# MERN To-Do CRUD App

This is a full-stack To-Do application built using the MERN stack.

## Features
- Add tasks
- View tasks
- Edit tasks
- Delete tasks

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Axios

## Run Locally

Backend
cd backend
npm install
node server.js

Frontend
cd frontend
npm install
npm start
